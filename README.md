# Obstacle Avoiding Robot Car
Arduino-based autonomous obstacle avoidance robot using ultrasonic sensors and motor drivers for real-time obstacle detection, autonomous navigation, collision avoidance, and dynamic path correction.
Implemented sensor-based movement control to improve navigation accuracy and robot mobility.

## Features
- Real-time obstacle detection
- Autonomous navigation
- Dynamic path correction
- Ultrasonic sensor integration
- Servo-based directional scanning
- Motor driver controlled movement

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- L293D Motor Driver Shield
- DC Motors
- Robot Chassis
- Battery Pack

## Technologies Used
- Arduino IDE
- Embedded C/C++
- Sensor Integration

## Working Principle
1. Ultrasonic sensor continuously measures distance.
2. Robot moves forward when path is clear.
3. If obstacle detected:
   - Robot stops
   - Scans left and right using servo motor
   - Compares distances
   - Chooses safer direction
